# Quick Start


Projekt wykorzystuje GitHub Actions.
Pipeline sprawdza:
- formatowanie (black)
- lint (pylint)
- testy (pytest)

Po poprawkach pipeline przechodzi poprawnie (status SUCCESS).

Dzieje sie wiele rzeczy zapraszam do obserwacji projektu
Użyta wersja Pythona: 3.14.5
